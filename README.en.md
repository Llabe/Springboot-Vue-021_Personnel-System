# SpringBoot-Vue-066springboot066人事系统

#### Description
基于 SpringBoot + Vue 的 B/S 架构人事管理系统，采用 SSM 框架实现核心业务逻辑。系统包含管理员和员工双角色权限控制，支持部门管理、员工考勤、上下班记录、加班管理、薪酬核算等核心人事功能，通过 MyBatis Plus 实现高效数据持久化操作。
